デバッグガイド

実装中によく遭遇するエラーとその解決方法をまとめました。
エラーが出たら、まずブラウザのコンソール（F12）とターミナル（サーバ側）を確認してください。

エラー1: CORS エラー

エラーメッセージ:
Access to XMLHttpRequest at 'http://localhost:3000' from origin 'http://127.0.0.1:5500' has been blocked by CORS policy

原因: Live Server（ポート5500）からNode.jsサーバ（ポート3000）にアクセスしている。異なるオリジン間の通信はブラウザがブロックする。

解決方法:

  • Live Serverを使わず、Node.jsサーバから直接HTMLを配信する
  • express.static() で public フォルダを公開し、http://localhost:3000 でアクセスする
JavaScript ── server.js
const express = require('express');
const app = express();

// publicフォルダの中身を静的ファイルとして配信
app.use(express.static('public'));

// http://localhost:3000 でindex.htmlが表示される

エラー2: Socket.IO 接続失敗

エラーメッセージ:
GET http://localhost:3000/socket.io/?... 404 (Not Found)

原因: Socket.IOがサーバ側で正しく設定されていない。Expressだけ起動してSocket.IOを紐づけていない。

解決方法: http.createServer でHTTPサーバを作り、Socket.IOをそこに紐づける。

JavaScript ── server.js
const express = require('express');
const http = require('http');
const { Server } = require('socket.io');

const app = express();
const server = http.createServer(app);
const io = new Server(server);

app.use(express.static('public'));

// app.listen() ではなく server.listen() を使う
server.listen(3000, () => {
  console.log('Server running on http://localhost:3000');
});

エラー3: 3Dモデルが表示されない

症状: glTFモデルの読み込みを指定しているが、3D空間に何も表示されない。

よくある原因と解決方法:

原因解決方法
ファイルパスが間違っている相対パスを確認。public/models/scene.glb なら src="models/scene.glb"
モデルのスケールが極端scale="0.01 0.01 0.01" のように調整する
モデルの位置がカメラから遠いposition="0 0 -5" のように手前に配置する
MIME typeエラー.glb ファイルが application/octet-stream で配信されているか確認

デバッグのコツ: まず<a-box>で同じ位置にテスト表示し、位置が正しいか確認してからモデルに置き換える。

エラー4: アバターが同期されない

症状: 他のユーザのアバターが表示されない、または動かない。

確認ポイント:

  • console.log でSocket.IOのイベントが発火しているか確認する
  • サーバ側で socket.broadcast.emit を使っているか（io.emit だと自分にも送信される）
  • クライアント側で受信したデータを正しくA-Frameエンティティに反映しているか
  • 位置データのフォーマットが送信側と受信側で一致しているか
JavaScript ── デバッグ用console.log
// クライアント側: 送信時に確認
socket.emit('move', posData);
console.log('送信:', posData);

// クライアント側: 受信時に確認
socket.on('moved', (data) => {
  console.log('受信:', data);
  // ここでA-Frameエンティティを更新
});

参考 よくある実装パターン: アバター同期

サーバ側（server.js）

JavaScript ── server.js（アバター同期部分）
const users = {};

io.on('connection', (socket) => {
  console.log('接続: ' + socket.id);

  // 既存ユーザ情報を新規ユーザに送信
  socket.emit('currentUsers', users);

  // 位置更新
  socket.on('move', (data) => {
    users[socket.id] = data;
    socket.broadcast.emit('userMoved', {
      id: socket.id,
      ...data
    });
  });

  // 切断
  socket.on('disconnect', () => {
    delete users[socket.id];
    io.emit('userLeft', socket.id);
    console.log('切断: ' + socket.id);
  });
});

クライアント側（app.js）

JavaScript ── public/app.js（アバター同期部分）
const socket = io();
const avatars = {};

// 自分の位置を定期送信
setInterval(() => {
  const camera = document.querySelector('[camera]');
  const pos = camera.getAttribute('position');
  const rot = camera.getAttribute('rotation');
  socket.emit('move', {
    position: { x: pos.x, y: pos.y, z: pos.z },
    rotation: { x: rot.x, y: rot.y, z: rot.z }
  });
}, 100);

// 他のユーザのアバターを表示・更新
socket.on('userMoved', (data) => {
  if (!avatars[data.id]) {
    const el = document.createElement('a-box');
    el.setAttribute('color', '#4FC3F7');
    el.setAttribute('scale', '0.5 1 0.3');
    document.querySelector('a-scene').appendChild(el);
    avatars[data.id] = el;
  }
  avatars[data.id].setAttribute('position', data.position);
  avatars[data.id].setAttribute('rotation', data.rotation);
});

// ユーザ退出時にアバター削除
socket.on('userLeft', (id) => {
  if (avatars[id]) {
    avatars[id].remove();
    delete avatars[id];
  }
});
参考 よくある実装パターン: チャット吹き出し 
JavaScript ── public/app.js（チャット吹き出し部分）
// チャット送信
document.getElementById('send-btn').addEventListener('click', () => {
  const input = document.getElementById('chat-input');
  const msg = input.value.trim();
  if (!msg) return;
  socket.emit('chat', msg);
  input.value = '';
});

// チャット受信 → 3D吹き出し表示
socket.on('chatMessage', (data) => {
  const avatar = avatars[data.id];
  if (!avatar) return;

  // 既存の吹き出しを削除
  const old = avatar.querySelector('.bubble');
  if (old) old.remove();

  // 新しい吹き出しを作成
  const bubble = document.createElement('a-text');
  bubble.setAttribute('value', data.message);
  bubble.setAttribute('position', '0 1.5 0');
  bubble.setAttribute('align', 'center');
  bubble.setAttribute('color', 'white');
  bubble.setAttribute('scale', '2 2 2');
  bubble.setAttribute('class', 'bubble');
  bubble.setAttribute('look-at', '[camera]');
  avatar.appendChild(bubble);

  // 5秒後に消す
  setTimeout(() => bubble.remove(), 5000);
});
