説明

静止した3D空間に動きを加えると、表現の幅が一気に広がります。
A-Frame の animation コンポーネントを使えば、HTMLの属性だけでオブジェクトを動かすことができます。

1.1 animation コンポーネントの基本

パラパラ漫画は、少しずつ違う絵を高速で切り替えて動きを表現します。A-Frame の animation も同じで、ある値を少しずつ変化させることで滑らかな動きを作ります。

animation コンポーネントは、オブジェクトの属性を時間とともに変化させる仕組みです。HTMLの属性として直接書けます。

HTML
<!-- 箱をY軸で360度回転させ続ける -->
<a-box position="0 1 -3" color="red"
       animation="property: rotation;
                  to: 0 360 0;
                  dur: 3000;
                  loop: true;
                  easing: linear"></a-box>
属性説明例
property変化させる対象のプロパティrotation, position, scale, color, opacity
from開始値（省略するとオブジェクトの現在値）0 0 0
to終了値0 360 0
dur所要時間（ミリ秒）3000（3秒）
loop繰り返し（true / false / 回数）true（無限）
easing加速・減速のパターンlinear, easeInOutQuad
directionアニメーションの方向normal, alternate, reverse
startEventsアニメーション開始のトリガーイベントclick, mouseenter

読み方（カタカナ）

animationアニメーション ── 動きをつける仕組み（A-Frame ではコンポーネント名）
propertyプロパティ ── 「特性・属性」の意味。何を変化させるかを指定
dur = durationデュレーション ── 「所要時間」。ミリ秒単位（1000 = 1秒）
requestAnimationFrameリクエストアニメーションフレーム ── 内部で毎フレーム値を更新する仕組み（A-Frame が肩代わりしてくれる）

具体例

dur: 3000 で「from の値から to の値まで 3秒かけて変化」。loop: true で「無限に繰り返し」。両方を組み合わせると 3秒で1周する回転がずっと続く。

1.2 アニメーションできるプロパティ

以下のプロパティをアニメーションで変化させられます。

property変化の内容値の書き方
position位置を移動するto: 2 3 -5（x y z）
rotation回転するto: 0 360 0（x y z 軸の角度）
scale拡大・縮小するto: 2 2 2（x y z の倍率）
material.color色を変えるto: #FF0000
material.opacity透明度を変える（0=透明、1=不透明）to: 0
light.intensityライトの明るさを変えるto: 0

opacity を使うときの注意

material.opacity をアニメーションするには、オブジェクトに material="transparent: true" を追加する必要があります。これがないと透明度が変化しません。

読み方（カタカナ）

positionポジション ── 「位置」。3D空間の x y z 座標
rotationローテーション ── 「回転」。x y z 軸まわりの角度（度）。Three.js でいう mesh.rotation の更新
scaleスケール ── 「倍率」。1=等倍、2=2倍に拡大、0.5=半分に縮小
material.opacityマテリアル ドット オパシティ ── 「材質の不透明度」。0〜1 の小数

具体例

property: rotation + to: 0 360 0 は「Y軸まわりに360度回す」。property: scale + to: 2 2 2 は「縦横奥行きすべて2倍に拡大」。プロパティ名はドット記法でネストもできる（material.color, material.opacity）。

1.3 イージング関数 ── 動きの「味付け」

ボールを投げると、最初は速く、頂点で一瞬止まり、また加速します。イージング関数は、こうした自然な動きを数学的に表現する仕組みです。
easing動き使いどころ
linear等速。一定の速度で動く。回転、ループアニメーション
easeInQuadゆっくり始まり加速する登場アニメーション
easeOutQuad速く始まり減速する着地、停止
easeInOutQuadゆっくり始まり、加速し、ゆっくり止まる自然な移動、汎用的
easeInOutElastic始めと終わりの両方でバネのように伸縮する注目を引くUI効果
easeOutElastic終わり際に勢いよく行き過ぎてバネのように揺れ戻る弾むUI、強調・登場の演出

読み方（カタカナ）

easingイージング ── 「ease=ゆるめる」から。動きのカーブのこと
Tween (TweenJS)トゥイーン ── 「between=2点間」から。A-Frame の easing は内部で TweenJS の式と同じカーブを使う
easeIn / easeOutイーズイン／イーズアウト ── 入り（始め）でゆっくり／出（終わり）でゆっくり
Quad / Elasticクアド／エラスティック ── 2乗カーブ／バネのように伸縮（ゴム）

具体例

linear は等速（時計の秒針）、easeOutElastic は終わり際に行き過ぎてバネのように揺れ戻る動き、easeInOutElastic は始めと終わりの両方でゴムが伸び縮みするような動き。同じ dur（合計時間）でも、easing が違うと「印象」が大きく変わる。

1.4 複数アニメーションの同時実行

1つのオブジェクトに複数のアニメーションを同時に適用するには、animation__2, animation__3 のようにサフィックス（接尾辞）を付けます。

HTML
<!-- 回転しながら上下に移動する箱 -->
<a-box position="0 1 -3" color="#4FC3F7"
       animation="property: rotation;
                  to: 0 360 0;
                  dur: 2000;
                  loop: true;
                  easing: linear"
       animation__2="property: position;
                     from: 0 1 -3;
                     to: 0 3 -3;
                     dur: 1500;
                     loop: true;
                     direction: alternate;
                     easing: easeInOutQuad"></a-box>
  • animation → 最初のアニメーション（回転）
  • animation__2 → 2つ目のアニメーション（上下移動）
  • animation__3, animation__4 ... と増やせる
  • サフィックスの名前は自由: animation__bounce, animation__spin などもOK

読み方（カタカナ）

animation__2アニメーション アンダーバー ツー ── 2つ目のアニメーション。__ がサフィックス区切り
サフィックス (suffix)語尾につける識別子。__bounce, __color など任意の名前OK
mixer (AnimationMixer)ミキサー ── 「混ぜる装置」。Three.js で複数アニメーションを同時再生する仕組み。A-Frame の複数 animation__N は概念的に同じ
GLTFLoaderジーエルティーエフ ローダー ── glTF（3Dモデル形式）の読み込み機能。モデル付属のアニメーションを mixer で再生する

具体例

1つの箱に animation（回転）と animation__2（上下移動）を書くと、同時に両方が再生される。それぞれの dur, easing, loop は独立に設定できる。これは Three.js の AnimationMixer で複数 clip を play() するのと同じ発想。

1.5 イベントトリガー ── クリックで動かす

エレベーターのボタンを押すとドアが開くように、ユーザの操作をきっかけにアニメーションを開始できます。

startEvents を指定すると、アニメーションは自動では始まらず、指定したイベントが発生したときに開始します。

HTML
<!-- クリックすると大きくなる球 -->
<a-sphere position="0 1.5 -4" color="#76B900"
          animation="property: scale;
                     to: 2 2 2;
                     dur: 500;
                     startEvents: click;
                     easing: easeOutElastic"></a-sphere>

<!-- マウスが乗ると色が変わる箱 -->
<a-box position="2 1 -4" color="#333"
       animation="property: material.color;
                  to: #FF5252;
                  dur: 300;
                  startEvents: mouseenter"
       animation__2="property: material.color;
                     to: #333;
                     dur: 300;
                     startEvents: mouseleave"></a-box>
イベント発生タイミング
clickオブジェクトをクリック（またはカーソルの fuse で選択）したとき
mouseenterカーソル/マウスがオブジェクトに乗ったとき
mouseleaveカーソル/マウスがオブジェクトから離れたとき

click イベントを使うには

A-Frame の 3D空間で click イベントを発生させるには、カメラの中に <a-cursor> を配置する必要があります。第6回で学んだカーソルがここで活きてきます。

読み方（カタカナ）

startEventsスタートイベンツ ── 「開始のきっかけ（複数可）」。指定したイベントが来るまでアニメは止まったまま
click / mouseenterクリック／マウスエンター ── ユーザ操作。Three.js でいう raycaster + イベントリスナの仕組み
GLTFLoader animation mixerジーエルティーエフローダー アニメーション ミキサー ── 外部モデルのアニメをイベントで play() する Three.js の仕組み。A-Frame の startEvents はこれを HTML 属性化したもの
<a-cursor>エー カーソル ── 3D空間内で click を発生させるためのカメラ子要素

具体例

startEvents: click を書くと、アニメーションは「クリックされるまで待機」状態になる。クリックすると 1回だけ再生（loop: true ならその後繰り返し）。mouseenter と mouseleave をペアで使うと「ホバー中だけ大きくなる」効果が作れる。

1.6 アニメーションの方向（direction）

direction動作使いどころ
normalfrom → to を毎回繰り返す（デフォルト）回転（0→360→0→360...）
alternatefrom → to → from → to ... と往復する上下移動、点滅、呼吸するような動き
reverseto → from の方向に動く逆再生の表現
HTML
<!-- 上下に往復する球（呼吸するような動き） -->
<a-sphere position="0 1 -4" color="#93D500"
          animation="property: position;
                     from: 0 1 -4;
                     to: 0 2.5 -4;
                     dur: 2000;
                     loop: true;
                     direction: alternate;
                     easing: easeInOutQuad"></a-sphere>

📖 キーワード

easing（イージング）アニメーションの加速・減速カーブ。linear は等速、easeOut は減速停止。自然な動きに欠かせない概念。
direction: alternateアニメーションの再生方向を毎回反転する。from→to→from→to... と往復動作になる。
startEventsアニメーション開始のトリガーとなるイベント名。省略すると即時開始。click や mouseenter を指定してインタラクティブにする。
animation__2（サフィックス）1つのオブジェクトに複数のアニメーションを付けるための命名規則。__ の後に任意の名前を付ける。

animation 属性の読み方

以下のコードを読み、どんな動きになるか確認しましょう。

HTML
<a-box position="0 1 -3" color="#FF5252"
       animation="property: rotation;
                  to: 0 360 0;
                  dur: 4000;
                  loop: true;
                  easing: linear"></a-box>
属性値意味
propertyrotation回転を変化させる
to0 360 0Y軸で360度（1周）回転
dur40004秒間かけて1周する
looptrue永遠に繰り返す
easinglinear等速で回転する

結果: 赤い箱がY軸を中心に4秒で1周し続ける。等速で一定速度で回転。

読みやすい書き換え版（JavaScript で setAttribute する場合）

A-Frame の属性文字列を JavaScript で組み立てるときは、1行に詰め込まずに中間変数とヘルパー関数を使うと読みやすくなる:

JavaScript
// ❌ 1行に if と式を詰め込んだ書き方（読みにくい）
box.setAttribute('animation', isSpinning ? 'property: rotation; to: 0 360 0; dur: 4000; loop: true; easing: linear' : '');

// ✅ 中間変数 + 関数定義で意図を明示
function buildSpinAnimation(durationMs) {
  const property = 'rotation';            // 変化させる対象
  const targetAngle = '0 360 0';         // 終了値（Y軸1周）
  const easingCurve = 'linear';          // 等速
  return `property: ${property}; to: ${targetAngle}; dur: ${durationMs}; loop: true; easing: ${easingCurve}`;
}

if (isSpinning) {
  const spinConfig = buildSpinAnimation(4000);
  box.setAttribute('animation', spinConfig);
} else {
  box.removeAttribute('animation');
}

ポイント: ① 三項演算子（1行 if）を if/else に展開、② プロパティ・目標値・イージングを中間変数に分離、③ アニメ生成を buildSpinAnimation() 関数に切り出して再利用可能に。

改造指示（すべて実行すること）

解答例

例題 例題1 やってみよう（dur / easing の変更）解答

1.1 のサンプル <a-box>（Y軸回転、dur: 3000, easing: linear）を改造して試した結果:

変更1: dur: 3000 → 500
  → 1周にかかる時間が 3秒 → 0.5秒 に短縮され、回転が約6倍速くなる。
  → ぐるぐる目が回るような高速回転になる。

変更2: easing: linear → easeInOutQuad
  → 0.5秒の各サイクルの中で、開始は遅く、中盤で加速、終盤で減速する。
  → ただし loop: true でつなげているため、一周ごとに「ためて出る」感が出る。
  → 等速回転(linear)と比べて「リズムのある」回転に見える。

ポイント: dur はミリ秒指定で、値が小さいほど速くなる（直感と逆になりやすい）。easing は「変化のカーブ」であり、合計時間（dur）は変えずに速度の配分を変える。回転のように「終わりがない」アニメーションは linear が自然で、停止や登場は easeOut/easeInOut 系が自然。

例題 例題2 やってみよう（クリックで色変化）解答

1.5 のサンプル <a-box>（mouseenter で material.color を #FF5252 に変える）の to: を変更:

変更前: to: #FF5252  （赤）
変更例: to: #FF9800  （オレンジ）

見た目: ダーク（#333）の箱にカーソルを乗せると、
        300ms かけて鮮やかなオレンジに変化し、
        離すと黒に戻る。赤より柔らかく目を引く印象。

ポイント: material.color は HEX（#RRGGBB）や色名（red, cyan）で指定可能。アニメーションは RGB の各成分を線形補間するので、補色（例: 青 → 黄）を指定すると途中で灰色を通る。意図した色変化にするには 明度・彩度が近い色同士を選ぶか、間に中継色を挟む（animation__2 で連鎖）と良い。
