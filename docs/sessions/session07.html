説明

静止した3D空間に動きを加えると、表現の幅が一気に広がります。
A-Frame の animation コンポーネントを使えば、HTMLの属性だけでオブジェクトを動かすことができます。

1.1 animation コンポーネントの基本 課題に出ます

パラパラ漫画は、少しずつ違う絵を高速で切り替えて動きを表現します。A-Frame の animation も同じで、ある値を少しずつ変化させることで滑らかな動きを作ります。

animation コンポーネントは、オブジェクトの属性を時間とともに変化させる仕組みです。HTMLの属性として直接書けます。

HTML
<!-- 箱をY軸で360度回転させ続ける -->
<a-box position="0 1 -3" color="red"
       animation="property: rotation;
                  to: 0 360 0;
                  dur: 3000;
                  loop: true;
                  easing: linear"></a-box>

🖐 やってみよう（1分）

dur を 500 に変えて回転速度の違いを確認し、次に easing を easeInOutQuad に変えてみよう。

Notionに記録: 速度と動きの違いを1文で書く
属性説明例
property変化させる対象のプロパティ 課題に出ますrotation, position, scale, color, opacity
from開始値（省略するとオブジェクトの現在値）0 0 0
to終了値0 360 0
dur所要時間（ミリ秒）3000（3秒）
loop繰り返し（true / false / 回数）true（無限）
easing加速・減速のパターンlinear, easeInOutQuad
directionアニメーションの方向normal, alternate, reverse
startEventsアニメーション開始のトリガーイベントclick, mouseenter

1.2 アニメーションできるプロパティ 課題に出ます

以下のプロパティをアニメーションで変化させられます。

property変化の内容値の書き方
position位置を移動するto: 2 3 -5（x y z）
rotation回転するto: 0 360 0（x y z 軸の角度）
scale拡大・縮小するto: 2 2 2（x y z の倍率）
material.color色を変えるto: #FF0000
material.opacity透明度を変える（0=透明、1=不透明）to: 0
light.intensityライトの明るさを変えるto: 0

opacity を使うときの注意 課題に出ます

material.opacity をアニメーションするには、オブジェクトに material="transparent: true" を追加する必要があります。これがないと透明度が変化しません。

1.3 イージング関数 ── 動きの「味付け」 課題に出ます

ボールを投げると、最初は速く、頂点で一瞬止まり、また加速します。イージング関数は、こうした自然な動きを数学的に表現する仕組みです。
easing動き使いどころ
linear等速。一定の速度で動く。回転、ループアニメーション
easeInQuadゆっくり始まり加速する登場アニメーション
easeOutQuad速く始まり減速する着地、停止
easeInOutQuadゆっくり始まり、加速し、ゆっくり止まる自然な移動、汎用的
easeInOutElasticバネのように伸縮しながら動く注目を引くUI効果
easeOutBounce着地時にバウンドするボールの落下、弾む表現

1.4 複数アニメーションの同時実行 課題に出ます

1つのオブジェクトに複数のアニメーションを同時に適用するには、animation__2, animation__3 のようにサフィックス（接尾辞）を付けます。

HTML
<!-- 回転しながら上下に移動する箱 -->
<a-box position="0 1 -3" color="#4FC3F7"
       animation="property: rotation;
                  to: 0 360 0;
                  dur: 2000;
                  loop: true;
                  easing: linear"
       animation__2="property: position;
                     from: 0 1 -3;
                     to: 0 3 -3;
                     dur: 1500;
                     loop: true;
                     direction: alternate;
                     easing: easeInOutQuad"></a-box>
  • animation → 最初のアニメーション（回転）
  • animation__2 → 2つ目のアニメーション（上下移動）
  • animation__3, animation__4 ... と増やせる
  • サフィックスの名前は自由: animation__bounce, animation__spin などもOK

1.5 イベントトリガー ── クリックで動かす 課題に出ます

エレベーターのボタンを押すとドアが開くように、ユーザの操作をきっかけにアニメーションを開始できます。

startEvents を指定すると、アニメーションは自動では始まらず、指定したイベントが発生したときに開始します。

HTML
<!-- クリックすると大きくなる球 -->
<a-sphere position="0 1.5 -4" color="#76B900"
          animation="property: scale;
                     to: 2 2 2;
                     dur: 500;
                     startEvents: click;
                     easing: easeOutElastic"></a-sphere>

<!-- マウスが乗ると色が変わる箱 -->
<a-box position="2 1 -4" color="#333"
       animation="property: material.color;
                  to: #FF5252;
                  dur: 300;
                  startEvents: mouseenter"
       animation__2="property: material.color;
                     to: #333;
                     dur: 300;
                     startEvents: mouseleave"></a-box>

🖐 やってみよう（1分）

クリックで色が変わるコードの to の色を好きな色に変えて試してみよう。

Notionに記録: 変更した色のコードと見た目を1文で書く
イベント発生タイミング
clickオブジェクトをクリック（またはカーソルの fuse で選択）したとき
mouseenterカーソル/マウスがオブジェクトに乗ったとき
mouseleaveカーソル/マウスがオブジェクトから離れたとき

click イベントを使うには 課題に出ます

A-Frame の 3D空間で click イベントを発生させるには、カメラの中に <a-cursor> を配置する必要があります。第6回で学んだカーソルがここで活きてきます。

1.6 アニメーションの方向（direction）

direction動作使いどころ
normalfrom → to を毎回繰り返す（デフォルト）回転（0→360→0→360...）
alternatefrom → to → from → to ... と往復する上下移動、点滅、呼吸するような動き
reverseto → from の方向に動く逆再生の表現
HTML
<!-- 上下に往復する球（呼吸するような動き） -->
<a-sphere position="0 1 -4" color="#93D500"
          animation="property: position;
                     from: 0 1 -4;
                     to: 0 2.5 -4;
                     dur: 2000;
                     loop: true;
                     direction: alternate;
                     easing: easeInOutQuad"></a-sphere>

📖 キーワード

easing（イージング）アニメーションの加速・減速カーブ。linear は等速、easeOut は減速停止。自然な動きに欠かせない概念。
direction: alternateアニメーションの再生方向を毎回反転する。from→to→from→to... と往復動作になる。
startEventsアニメーション開始のトリガーとなるイベント名。省略すると即時開始。click や mouseenter を指定してインタラクティブにする。
animation__2（サフィックス）1つのオブジェクトに複数のアニメーションを付けるための命名規則。__ の後に任意の名前を付ける。

例題

animation 属性の読み方

以下のコードを読み、どんな動きになるか確認しましょう。

HTML
<a-box position="0 1 -3" color="#FF5252"
       animation="property: rotation;
                  to: 0 360 0;
                  dur: 4000;
                  loop: true;
                  easing: linear"></a-box>
属性値意味
propertyrotation回転を変化させる
to0 360 0Y軸で360度（1周）回転
dur40004秒間かけて1周する
looptrue永遠に繰り返す
easinglinear等速で回転する

結果: 赤い箱がY軸を中心に4秒で1周し続ける。等速で一定速度で回転。

コード体験 1

基本アニメーション ── animation-basic.html

S:\Web_Prog\Web07 フォルダを作成し、以下のファイルを作ります。

  1. VS Code で 「ファイル」→「フォルダーを開く」 から S:\Web_Prog\Web07 を開く
  2. 新しいファイル animation-basic.html を作成
  3. 下のコードをコピーして貼り付け、保存
HTML ── animation-basic.html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>Basic Animations</title>
  <script src="https://aframe.io/releases/1.7.1/aframe.min.js"></script>
</head>
<body>
  <a-scene>
    <!-- ライト -->
    <a-light type="ambient" color="#444"></a-light>
    <a-light type="directional" position="-1 2 1" intensity="0.6"></a-light>

    <!-- 1. 回転する箱 -->
    <a-box position="-3 1 -5" color="#FF5252"
           animation="property: rotation;
                      to: 0 360 0;
                      dur: 3000;
                      loop: true;
                      easing: linear"></a-box>

    <!-- 2. 上下にバウンドする球 -->
    <a-sphere position="0 1.5 -5" radius="0.8" color="#4FC3F7"
             animation="property: position;
                        from: 0 1.5 -5;
                        to: 0 3.5 -5;
                        dur: 1000;
                        loop: true;
                        direction: alternate;
                        easing: easeOutBounce"></a-sphere>

    <!-- 3. 点滅する円柱（透明度アニメーション） -->
    <a-cylinder position="3 1 -5" radius="0.5" height="2"
               color="#FFB800"
               material="transparent: true; opacity: 1"
               animation="property: material.opacity;
                          from: 1; to: 0.1;
                          dur: 1500;
                          loop: true;
                          direction: alternate;
                          easing: easeInOutQuad"></a-cylinder>

    <!-- 4. 拡大縮小するトーラス（2つのアニメーション同時） -->
    <a-torus position="0 1 -3" radius="0.5" radius-tubular="0.1"
            color="#93D500"
            animation="property: rotation;
                       to: 360 360 0;
                       dur: 4000;
                       loop: true;
                       easing: linear"
            animation__2="property: scale;
                          from: 1 1 1;
                          to: 1.5 1.5 1.5;
                          dur: 2000;
                          loop: true;
                          direction: alternate;
                          easing: easeInOutQuad"></a-torus>

    <!-- 床 -->
    <a-plane position="0 0 -5" rotation="-90 0 0"
             width="14" height="14" color="#333"></a-plane>

    <!-- 空 -->
    <a-sky color="#111"></a-sky>
  </a-scene>
</body>
</html>

  1. ファイルを保存（Ctrl+S / Cmd+S）
  2. エディタ上で右クリック → 「Open with Live Server」
  3. 4つのオブジェクトの動きを観察する

発展課題

アニメーションシーンを作る

VS Code で Web07 フォルダに animated-scene.html を新規作成し、アニメーションを活用した動きのある3Dシーンを作ってください。

必須要件

  • 5つ以上のオブジェクトにアニメーションを設定する
  • 3種類以上のプロパティをアニメーションする（rotation, position, scale, color, opacity から選択）
  • 2種類以上のイージングを使い分ける
  • 少なくとも1つのイベントトリガー（startEvents: click / mouseenter）を使う
  • 少なくとも1つの複数アニメーション同時実行（animation__2）を使う
  • テーマを決める（例: 「太陽系」「時計」「遊園地」「海の中」「ダンスフロア」など）

トレース トレース演習 1 解答
オブジェクト形何が変化するか（property）どう動くか速さ（dur）
A球（sphere）position（位置）y=1.5 から y=3 へ上下に往復移動（direction: alternate）。バウンドするような動き（easeOutBounce）。1000ms（1秒で片道）
B箱（box）rotation（回転）X軸とY軸の両方で360度回転。斜めに複雑な回転をする。等速（linear）。5000ms（5秒で1周）
C円柱（cylinder）material.opacity（透明度）不透明（1）から半透明（0）へ徐々に消え、また戻る。点滅するように見える。2000ms（2秒で片道）
トレース トレース演習 2 解答
ユーザ操作発生イベント何が起こるか
カーソルを箱に合わせるmouseenter箱が 1.5倍に拡大する（scale: 1 1 1 → 1.5 1.5 1.5）。easeOutElastic でバネのように弾みながら大きくなる。0.4秒かけて拡大。
カーソルを箱から外すmouseleave箱が元のサイズに戻る（scale: 1 1 1）。0.3秒で縮小。
箱をクリックするclick箱がY軸で360度回転する。0.8秒で1回転。
体験 コード体験 1 解答（記録例）
  • Q1: (1) 赤い箱がY軸で等速に回転し続けている。(2) 青い球が上下にバウンドしている（着地時に跳ねるような動き）。(3) 黄色い円柱が明滅（フェードイン・フェードアウト）を繰り返している。(4) 緑のトーラスが斜めに回転しながら拡大・縮小を繰り返している。
  • Q2: トレース演習1では球・箱・円柱の3つを予測したが、実際のコードのオブジェクト配置が少し異なっていた。動きの種類（回転・移動・透明度変化）は予測どおりだった。
  • Q3: easeOutBounce から easeInOutQuad に変えると、バウンドする「跳ね返り」がなくなり、滑らかに上下する動きになった。
体験 コード体験 2 解答（記録例）
  • Q1: (1) 赤い箱: クリックするとY軸で1回転する。(2) 青い球: カーソルを乗せると弾むように拡大し、離すと元に戻る。(3) 緑の円柱: クリックすると赤色に変わり、同時に上にジャンプする。
  • Q2: トレース演習2の予測とほぼ合っていた。easeOutElastic の「バネ感」が想像より強く、拡大時に一瞬大きくなりすぎてから戻る動きが特徴的だった。
  • Q3: はい、同時に起こった。animation と animation__2 の両方に startEvents: click が設定されているため、1回のクリックで色変化とジャンプが同時に発生した。
