説明

3Dシーンの見た目を決めるのは、オブジェクトの形や色だけではありません。
ライト（光源）とカメラ（視点）の設定で、同じオブジェクトでもまったく違う印象になります。

1.1 ライトの種類 課題に出ます

部屋全体を照らす蛍光灯、手元だけを照らすデスクライト、舞台のスポットライト ── 照明の種類で空間の雰囲気がまったく変わります。A-Frame の <a-light> も同じように、用途に合わせて使い分けます。

A-Frame では <a-light type="..."> でライトの種類を指定します。何もライトを置かない場合、A-Frame はデフォルトで ambient light（強度 0.39）と directional light（強度 0.39）を自動で追加します。自分で <a-light> を1つでも追加すると、デフォルトライトは無効になります。

type名称特徴用途
ambient環境光シーン全体を均一に照らす。影を作らない。方向がない。最低限の明るさを確保する下地の光
directional平行光源太陽光のように一方向から平行に照らす。影を作れる。太陽光、月光の再現
point点光源電球のように全方向に光を放つ。距離で減衰する。電球、ランプ、炎
spotスポットライト円錐状に光を当てる。角度と減衰を制御できる。舞台照明、懐中電灯
hemisphere半球光上（空）と下（地面）の2色で照らす。影は作らない。屋外の自然な雰囲気
HTML
<!-- 環境光: シーン全体をうっすら照らす -->
<a-light type="ambient" color="#444"></a-light>

<!-- 平行光源: 上方から太陽のように照らす -->
<a-light type="directional" color="#FFF" intensity="0.6"
         position="-1 2 1"></a-light>

<!-- 点光源: 電球のように光を放つ -->
<a-light type="point" color="#FFE082" intensity="0.8"
         distance="10" position="0 3 -4"></a-light>

<!-- スポットライト: 円錐状に照らす -->
<a-light type="spot" color="#FFF" intensity="1"
         angle="45" penumbra="0.2"
         position="0 4 -3" target="#stage-obj"></a-light>

主な属性:

属性説明デフォルト
intensity光の強さ（0 = 消灯、1 = 標準）1.0
color光の色#FFF（白）
distancepoint/spot の光が届く距離（0 = 無限）0
anglespot の照射角度（度）60
penumbraspot の境界ぼかし（0-1）0

1.2 影（シャドウ） 課題に出ます

現実世界では光があれば必ず影ができます。3D空間では影の計算はコストが高いため、明示的にONにしないと影は表示されません。

A-Frame で影を表示するには 3つの設定 がすべて必要です。

#設定コード
1シーンでシャドウを有効化<a-scene shadow="type: pcfsoft">
2ライトで影を投射（directional または spot のみ）<a-light ... castShadow="true">
3オブジェクトで影を投射/受け取りshadow="cast: true; receive: true"
HTML
<a-scene shadow="type: pcfsoft">
  <!-- 影を落とすライト -->
  <a-light type="directional" position="-1 3 1"
           intensity="0.6" castShadow="true"></a-light>

  <!-- 影を投射するオブジェクト -->
  <a-box position="0 1 -3" color="red"
         shadow="cast: true"></a-box>

  <!-- 影を受け取る床 -->
  <a-plane position="0 0 -4" rotation="-90 0 0"
           width="10" height="10" color="#CCC"
           shadow="receive: true"></a-plane>
</a-scene>

影が表示されない場合のチェックリスト 課題に出ます

  • <a-scene shadow="type: pcfsoft"> を書いたか？
  • ライトの castShadow="true" を指定したか？
  • ライトの type は directional か spot か？（ambient / point / hemisphere は影を作れない）
  • オブジェクトに shadow="cast: true" を書いたか？
  • 床に shadow="receive: true" を書いたか？

1.3 カメラとコントロール 課題に出ます

映画の撮影では、カメラの位置や向きで同じシーンでも印象がまったく変わります。3D空間でも、カメラの位置を変えると見える景色が変わります。

A-Frame ではデフォルトでカメラが position="0 1.6 0"（人間の目の高さ）に自動配置されます。<a-camera> を自分で書くと、デフォルトカメラは無効になります。

要素/属性説明
<a-camera>カメラエンティティ。position で初期位置を指定する。
look-controlsマウスドラッグやタッチで視線の方向を変える（デフォルトON）
wasd-controlsW/A/S/Dキーでカメラの位置を移動する（デフォルトON）
fov視野角（Field of View）。大きいと広角、小さいと望遠。デフォルト: 80
HTML
<!-- カメラを高い位置から見下ろすように配置 -->
<a-camera position="0 5 8" rotation="-30 0 0"></a-camera>

<!-- 移動を無効にして固定カメラにする -->
<a-camera position="0 1.6 4"
          look-controls="enabled: false"
          wasd-controls="enabled: false"></a-camera>

1.4 カーソル（a-cursor） 課題に出ます

<a-cursor> はカメラの中央から前方にレイキャスト（光線）を飛ばし、3Dオブジェクトとの交差判定を行う要素です。VR空間でのクリック操作に使います。

HTML
<a-camera position="0 1.6 0">
  <!-- カメラの子要素としてカーソルを配置 -->
  <a-cursor color="#FF5252" fuse="true" fuse-timeout="1500"></a-cursor>
</a-camera>
属性説明
fusetrue にすると、一定時間見つめるだけでクリック扱いになる（VR向け）
fuse-timeoutfuse でクリックが発生するまでの時間（ミリ秒）
colorカーソルの表示色

カーソルのイベント: カーソルがオブジェクトと交差すると mouseenter、離れると mouseleave、クリックすると click イベントが発生します。第7回のアニメーションと組み合わせて使います。

1.5 複数カメラの切り替え（概念）

A-Frame では複数の <a-camera> を配置し、active 属性で切り替えることができます。

HTML
<!-- 通常の一人称視点 -->
<a-camera id="cam-fps" position="0 1.6 0" active="true"></a-camera>

<!-- 上からの俯瞰カメラ -->
<a-camera id="cam-top" position="0 10 0" rotation="-90 0 0"
          active="false"
          look-controls="enabled: false"
          wasd-controls="enabled: false"></a-camera>

JavaScript で setAttribute('active', true/false) を切り替えてカメラを変更します。詳細は後の回で扱います。

例題

ライトの有無による見え方の違いを読む

以下の3パターンのコードで、同じ白い球がどう見えるか確認しましょう。

HTML
<!-- パターンA: ライトなし（A-Frameデフォルト） -->
<a-scene>
  <a-sphere position="0 1.5 -4" color="white"></a-sphere>
  <a-sky color="#222"></a-sky>
</a-scene>

<!-- パターンB: 環境光のみ -->
<a-scene>
  <a-light type="ambient" color="#FFF" intensity="0.5"></a-light>
  <a-sphere position="0 1.5 -4" color="white"></a-sphere>
  <a-sky color="#222"></a-sky>
</a-scene>

<!-- パターンC: 環境光 + 平行光源 -->
<a-scene>
  <a-light type="ambient" color="#FFF" intensity="0.3"></a-light>
  <a-light type="directional" color="#FFF" intensity="0.8"
           position="-1 2 1"></a-light>
  <a-sphere position="0 1.5 -4" color="white"></a-sphere>
  <a-sky color="#222"></a-sky>
</a-scene>
パターン使用ライト球の見え方
Aなし（デフォルト自動追加）明るい部分と暗い部分ができる（デフォルトの ambient + directional）
Bambient のみ全体が均一な灰白色。立体感がなく平面的に見える。
Cambient + directional光の当たる面は明るく、反対側は暗い。立体感がある。

トレース トレース演習 1 解答
ライト種類色予想される効果影を作るか？
1つ目ambient（環境光）#333（暗い灰色）シーン全体をうっすら照らす。立体感は出ない。最低限の明るさを提供する下地。いいえ
2つ目directional（平行光源）#FFA500（オレンジ）右上（position="2 4 1"）からオレンジ色の平行光が当たる。オブジェクトの右上が明るく、左下が暗くなる。強度0.7。はい（castShadow="true"）
3つ目point（点光源）#4FC3F7（水色）左側（position="-2 2 -3"）から水色の光が放射される。距離8まで届く。オブジェクトの左側が青みがかる。いいえ（point light はA-Frameで影を投射しない設定）

追加の問い: はい、箱の影は床に表示されます。理由: (1) <a-scene shadow="type: pcfsoft"> でシャドウが有効、(2) directional light に castShadow="true" が設定されている、(3) 箱に shadow="cast: true"、床に shadow="receive: true" が設定されている。3つの条件がすべて揃っているためです。

トレース トレース演習 2 解答
カメラ設定カメラの位置と向き見え方
Aposition="0 1.6 0"（デフォルト）人間の目の高さ（1.6m）から正面を見る。箱は z=-4 にあるので、4m 先に赤い箱が見える。
Bposition="0 10 -4" rotation="-90 0 0"箱の真上10mの位置から真下を見下ろす俯瞰視点。箱の上面だけが見える。
Cposition="5 1.6 -4"箱と同じ z=-4 だが、右に5mずれた位置。カメラはデフォルトで z=-方向を向くため、箱は視界の左端に見えるか、視界外になる可能性がある。
体験 コード体験 1 解答（記録例）
  • Q1: 3つのオブジェクトはすべて白色だが、左側の球は点光源（暖色）に近いため少し黄色みがかって見えた。右側の円柱は directional light（白）で照らされた面が最も明るかった。オブジェクトの裏側は暗い。
  • Q2: ambient light を削除すると、光が当たっていない面が完全に真っ暗になった。ambient light はシーン全体の最低限の明るさを保つ役割があることがわかった。
体験 コード体験 2 解答（記録例）
  • Q1: 床に影が表示された。directional light が左上（position="-2 4 2"）から照らしているため、影は各オブジェクトの右下方向に伸びていた。
  • Q2: カーソル（緑の小さな円）は画面中央に表示されていた。カーソルをオブジェクトに合わせると、カーソルがオブジェクトの表面に「くっつく」ように見えた（レイキャストが交差した位置）。
  • Q3: カメラを "0 8 -4" + rotation="-90 0 0" にすると真上からの俯瞰視点になり、3つのオブジェクトの上面と床の影を一度に見渡せた。

標準 標準課題 1 解答（ライト・影・カメラの知識）

問1.

type名称（日本語）光の特徴影を作れるか？
ambient環境光シーン全体を均一に照らす。方向がない。いいえ
directional平行光源太陽光のように一方向から平行に照らす。はい
point点光源電球のように全方向に光を放つ。距離で減衰する。いいえ（A-Frameでは非対応）
spotスポットライト円錐状に光を当てる。角度と減衰を制御できる。はい
hemisphere半球光上（空）と下（地面）の2色で照らす。いいえ

問2. 影を表示するための3つの設定:

  1. シーンでシャドウを有効化: <a-scene shadow="type: pcfsoft">
  2. ライトで影を投射: <a-light type="directional" castShadow="true">（type は directional または spot）
  3. オブジェクトで影を設定: 影を落とすオブジェクトに shadow="cast: true"、影を受け取る面に shadow="receive: true"

問3.

#文正誤訂正
Aambient light は影を作ることができる。誤りambient light は方向がないため影を作れません。影を作れるのは directional と spot のみです。
B自分で <a-light> を追加すると、デフォルトライトは無効になる。正しい--
Cカメラの wasd-controls はマウスで視線を動かす機能である。誤りwasd-controls はキーボードのW/A/S/Dキーでカメラの位置を移動する機能です。マウスで視線を動かすのは look-controls です。
D<a-cursor fuse="true"> は一定時間見つめるとクリック扱いになる。正しい--

問4. 1.6 はメートル単位で、人間の平均的な目の高さ（約160cm）を表しています。3D空間で一人称視点の体験を作る際、現実世界と同じ視点の高さにすることで、自然な没入感を得られるようにしています。
